Map: Features
=============

OpenWISP provides advanced map functionality to visualize, locate, and
track devices across geographic and indoor maps.

- **Enhanced Device Popups:** Map popups include search and filtering
  options based on device monitoring status.

  .. image:: gifs/Popup-filter.gif
      :alt: Popup filter

- **Indoor Map View with Floor Navigation:** The dashboard map includes an
  indoor view with floor switching and fullscreen mode.

  .. image:: gifs/Indoor-map-view.gif
      :alt: Indoor map view

- **Location View from Device Details Page:** The device detail page
  provides navigation to a full-page geographic map for the associated
  location.

  .. image:: gifs/View-on-general-map-button.gif
      :alt: Location View from Device Details Page

- **Indoor Device View from Device Details Page:** The device detail page
  provides navigation to a full-page indoor map showing the device
  position.

  .. image:: gifs/View-on-indoor-map-button.gif
      :alt: Indoor Device View from Device Details Page

- **Shareable Geographic Map URLs:** Interactions with location or device
  nodes on the geographic map update the URL.

  .. image:: gifs/Bookmark-url-map.gif
      :alt: Shareable Geographic Map URLs

- **Shareable Indoor Map URLs:** Interactions with indoor device nodes
  update the URL.

  .. image:: gifs/Bookmark-url-indoor-map.gif
      :alt: Shareable Geographic Map URLs

- **Real-Time Device Position Updates:** Device positions on the map are
  updated when new location data is received.

  .. image:: gifs/Moving-devices.gif
      :alt: Real-Time Device Position Updates
